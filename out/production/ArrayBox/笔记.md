## 数组是JAVA开发者创建好的类型

1. 概述

- 数组是一个小容器 存储一组元素

添加元素 获取元素 删除元素

- 创建一个数组对象

```java

    int[] array = new int[10];

```

- 添加元素

```java

    array[0] = 10//根据数组的索引添加元素

```

- 获取元素

```java

int value = array[0]//拿到数组索引为0号元素

```

- 遍历

```java
    for (int i = 0;i < array.length;i++>){
        System.out.print(array[i]);
    }
```
